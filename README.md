# 路由权限

## 服务端

```bash

npm i -g koa-generator
koa2 [project-name]
cd [project-name]
npm install
npm i koa2-cors -S
npm run dev
```

## 路由权限

1. uid -> 后端 API - 路由权限 API

2. 后端 -> 用户对路由权限列表 -> 前端 -> JSON

3. JSON -> 树形结构化

4. 树形结构化数据 -> vue 路由结构

5. 路由结构动态 -> 静态路由

6. 树形结构化的数据 -> 菜单组件
